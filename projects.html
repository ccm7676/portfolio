MORSY

Morsy is an open-source private messenger. Many private messengers nowadays have one major fault, which is the centralized nature of the servers which the messages are sent through. Morsy's primary aim is to provide privacy to its users while requiring zero trust in the servers. This is achieved by using WebRTC to send the messages directly between users, making the server's sole purpose to provide the initial connection.

KENATOR

Kenator is a simple diabetes calculator. It is designed to simplify the process of calculating the amount of insulin units needed to be injected for a given amount of food. It has a modern and clearly labeled user interface and web based, which makes it accessible on almost any device.

SUDOKUH

SudoKuh is a Sudoku app written in python. Unlike many other Sudoku apps, SudoKuh doesn't have preprogrammed puzzles, instead it generates them using a custom algorithm with varying difficulties.
